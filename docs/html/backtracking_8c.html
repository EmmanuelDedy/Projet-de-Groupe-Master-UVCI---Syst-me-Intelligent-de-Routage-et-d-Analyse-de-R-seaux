Recherche par Backtracking avec Élagage (Pruning).

Explore l'arbre des possibilités pour trouver un chemin satisfaisant des contraintes multiples (Multi-Constrained Path Problem).

Contraintes traitées :

  • Bande passante >= min_bp
  • Coût total <= max_cout
  • Passage obligatoire par une liste de nœuds (Waypoints)

Chemin* Le meilleur chemin valide trouvé.
